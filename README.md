# mytomcat
本项目是基于Netty实现自己的web容器，重复造轮子，来巩固相关知识点
**功能**：
   + [x] **IOC容器**：通过自定注解`@Bee`实现对所有对象的管理,`通过@Autowired`注解实现对象的依赖注入
   + [x] **请求分发** 使用`@Controller`和`@RequestMappin` 实现自定义路由，匹配对应请求
   + [ ] 自动参数转换
   + [ ] 结果渲染
   + [ ] 前置后置拦截器
   + [ ] session管理
   + [ ] cookie管理
   + [ ] 结果渲染
